<strong>

The <strong> tag is used to show something has strong importance.

<em>

The <em> tag is used to create emphasis on certain words.

<blockquote>

The <blockquote> tag is used for quotes that are longer than others and you want to seperate out from the containing tag. This text is wrapped in a <blockquote> tag.

<q>

The <q> tag is used for quotes in larger bodies of text, such as in a <p> tag. This text is wrapped in a <q> tag.

<abbr>

The <abbr> tag is used to show abbreviations, such as FSU

<cite>

The <cite> tag is used for citations of book or article titles, such as Game of Thrones, by George R. R. Martin.

<dfn>

The <dfn> tag is used to show the defning instance of a term or concept.

<address>

The <address> tag is used to show an address.

1234 Sesame St.

<ins> and <del>

The <ins> tag is used to show additions to text, while the <del> tag is used to show subtractions from text.

<s>

The <s> tag is used to show information that is no longer relevant, but should not be deleted.

<ol>

The <ol> tag is used to create a numerically ordered list.

  1. <ol>Inside the ordered list would be an <li>, or list item.
  2. <li>Each <li> creates another item in the list, and is wrapped in an li tag like this line.</li>
  3. These are in the proper numerical order that will be displayed.</ol>

<ul>

The <ul> tag is used to create a unordered list.

  • <ol>Inside the unordered list would be an <li>, or list item.
  • <li>Each <li> creates another item in the list, and is wrapped in an li tag like this line.</li>
  • These are in the order that will be displayed.</ol>
